# Language-Specific SEO Guidelines

This document provides SEO optimization guidelines for each supported language.

---

## Italian (it)

### Character Limits
| Element | Optimal | Maximum |
|---------|---------|---------|
| Title | 50-60 chars | 70 chars |
| Description | 150-160 chars | 180 chars |

### Keyword Research Tips
- Use Google.it for trend analysis
- Italian users search with articles ("il", "la", "i", "le")
- Consider regional variations (North vs South Italy)

### Tone & Style
- **Formality:** Professional but accessible (professionale ma accessibile)
- **Pronouns:** Use "tu" (informal) for B2C, "Lei" (formal) for B2B
- **Preferred:** Direct, clear communication

### Cultural Notes
- Italians value design aesthetics highly
- Visual content is important
- Trust local references over international ones
- Prefer localized content over obvious translations

### Common Keyword Patterns
| English | Italian (High Volume) | Notes |
|---------|----------------------|-------|
| AI image generator | creare immagini con AI | Action-oriented preferred |
| free | gratis / gratuito | "gratis" more common |
| best | migliore / top | "migliore" more formal |
| how to | come | Standard question format |

## Japanese (ja)

### Character Limits
| Element | Optimal | Maximum |
|---------|---------|---------|
| Title | 25-35 chars | 40 chars |
| Description | 70-100 chars | 120 chars |

### Keyword Research Tips
- Use "イラスト" (illustration) rather than "画像" (image) for creative contexts
- Katakana for foreign terms: "AI" = "AI" or "エーアイ"
- Mix of kanji, hiragana, katakana improves readability

### Tone & Style
- **Formality:** Formal (です/ます form) for UI text
- **Pronouns:** Often omitted; when used, polite forms
- **Preferred:** Concise, respectful

### Cultural Notes
- Anime/manga style resonates well
- High attention to detail expected
- Quality over quantity
- Trust signals very important

### Common Keyword Patterns
| English | Japanese (High Volume) | Notes |
|---------|----------------------|-------|
| AI image generator | AI イラスト生成 | Not "AI画像生成器" |
| create images | 画像を作る / イラスト作成 | Context-dependent |
| free | 無料 | Standard |
| professional | プロ / プロフェッショナル | Katakana common |

## General Best Practices

### 1. Always Research, Never Translate
```
English keyword → Research local expressions → Validate with Google Trends → Use highest volume
```

### 2. Preserve Brand Voice
- Maintain consistent tone across languages
- Adapt, don't just translate
- Keep core value proposition intact

### 3. Technical SEO Consistency
- Implement hreflang tags correctly
- Maintain consistent URL structure
- Use proper character encoding (UTF-8)

### 4. Cultural Adaptation
- Understand local user behavior
- Adapt examples and references
- Consider local competitors

### 5. Quality Assurance
- Native speaker review when possible
- A/B test localized content
- Monitor search performance by locale
