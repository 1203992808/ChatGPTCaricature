description: 本地关键词研究和优化
argument-hint: [--city <name>] [--analyze]
---

生成和优化本地关键词。生成"城市+服务"关键词组合、分析搜索量和竞争度、提供页面优化建议。

## 参数
- `--city`: 指定城市名称
- `--analyze`: 分析现有本地关键词使用情况

## 关键词类型
- "Service + City"
- "Service + near + City"
- "Best + Service + in + City"
- "Service + Neighborhood"
- "Service + [zip code]"
