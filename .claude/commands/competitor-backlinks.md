---
description: 竞争对手外链分析
argument-hint: [competitor-url] [--deep]
---

分析竞争对手的外链质量和数量，识别高价值的外链来源，发现外链差距，建议获取相同外链的策略。

## 参数

- `$1`: 竞争对手网站 URL
- `--deep`: 执行深度分析（包括锚文本分析）

## 使用示例

```bash
/competitor-backlinks https://competitor.com
```

分析竞争对手的外链来源并提供策略建议。

```bash
/competitor-backlinks https://competitor.com --deep
```

包含锚文本分析和详细的外链质量评估。
