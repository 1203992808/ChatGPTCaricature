对项目进行 E-E-A-T（Experience, Expertise, Authoritativeness, Trustworthiness）权威性审计。评估专业度展示、专业能力、权威性建立和信任度建设。

## 参数

- `--detailed`: 生成详细的改进建议报告

## E-E-A-T 维度

1. **Experience（经验）** - 案例研究、实际经验分享、内容深度
2. **Expertise（专业能力）** - 技术准确性、作者专业性、领域知识
3. **Authoritativeness（权威性）** - 外部引用、内容独特性、社交媒体影响力
4. **Trustworthiness（信任度）** - 更新频率、数据来源、透明度

## 使用示例

```bash
/ee-audit
```

生成 E-E-A-T 审计报告和评分。

```bash
/ee-audit --detailed
```

包含详细的改进建议和行动清单。
